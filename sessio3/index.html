    Sessió 3: Organitzar el contingut amb seccions i disseny en caixes¶

    Objectius¶

    • Aprendre a estructurar una pàgina amb etiquetes semàntiques: <header>, <section>, <footer>.
    • Entendre què és un <div> i com s'utilitza per agrupar elements.
    • Aplicar estils CSS a seccions diferents de la web.
    • Millorar la llegibilitat i organització del codi HTML.

    Eines recomanades¶

    • CodePen: https://codepen.io/

    Contingut¶

    Quan creem una pàgina web, no sols importa el contingut, sinó també com l'organitzem. HTML ens permet estructurar el contingut usant "caixes" i etiquetes amb significat, per tal que siga més llegible, ordenat i professional.

    Què és una caixa en HTML?¶

    Una caixa és una agrupació d'elements. Pot ser genèrica (amb <div>) o amb significat (amb etiquetes semàntiques com <header>, <section>, etc.). A aquestes caixes els podem aplicar estils CSS, com fons, mides, colors...

    🔎 Diferència entre div i etiquetes semàntiques¶

    Element Què fa Quan usar-lo
    <div> Caixa genèrica, sense significat propi Quan no hi ha una etiqueta semàntica adequada
    <header> Capçalera d'una secció o de tota la web Per a posar títols, menús, logos...
    <section> Secció de contingut relacionada Per a dividir el contingut en blocs
    <footer> Peu de pàgina Per a posar crèdits, contactes, copyright
    <main> Contingut principal Per al contingut central de la web
    <article> Contingut autònom (ex: una entrada) Per blocs reutilitzables de contingut

    Avantatges d'usar etiquetes semàntiques¶

    • Millor organització visual i lògica.
    • Facilita l'accessibilitat i lectura per part de navegadors i motors de cerca.
    • Millor mantenibilitat del codi.
    • Millor experiència d'usuari i posicionament (SEO).

    Exemple visual d'una web bàsica estructurada¶

    <body>
  <header>
    <h1>Benvingut al meu lloc web</h1>
  </header>

  <main>
    <section id="sobre">
      <h2>Sobre mi</h2>
      <p>Sóc l'alumna Júlia i m'encanta la ciència!</p>
    </section>

    <section id="projectes">
      <h2>Projectes</h2>
      <p>Aquí poso coses que he fet a classe.</p>
    </section>
  </main>

  <footer>
    <p>© 2025 Júlia Web</p>
  </footer>
</body>

    Activitat¶

    1. Fes una secció de la teua pàgina web amb un <header> que continga un títol en <h1> i una frase breu amb l'etiqueta <p> de la teua pàgina.
    2. Afegeix un <main> amb dues seccions <section>:
    3. Una, amb un id "hobby", per a mostrar la teua afició preferida amb la seua imatge associada i el link a la teua afició.
    4. Una altra, amb un id "sobre" que continga un paràgraf amb una breu descripció sobre eixa afició, així com un enllaç a una pàgina relacionada.
    5. Afegeix un <footer> amb el teu nom i cognoms i una adreça de contacte (correu electrònic o xarxes socials).
    6. Dins del header, afigeix un <nav> amb un menú d'enllaços a les seccions de la teua pàgina.
    7. Per a enllaçar a les seccions, utilitza l'atribut href amb el valor #id_de_la_secció.
    8. Fes que els enllaços dins del nav estiguen separant utilitzant una barra vertical (|) i posal's en la mateixa linia.
    9. Abans d'acabar fes que la web una coherencia temàtica i que parle d'un mateix tema en totes les seccions.
